MantiLoad Translation Files
===========================

This directory contains translation files for MantiLoad plugin.

Current Status:
- mantiload.pot: Translation template with 318 translatable strings
- Fully i18n ready for WordPress.org submission
- All admin menu items, notices, and UI strings are translatable

Files:
------
mantiload.pot - POT template file for translators

How to Translate:
-----------------
1. Use Poedit or similar tool to open mantiload.pot
2. Create translations for your language (e.g., mantiload-fa_IR.po for Persian)
3. Save both .po and .mo files in this directory
4. WordPress will automatically load the translations

For WordPress.org:
------------------
Translations can also be contributed via:
https://translate.wordpress.org/projects/wp-plugins/mantiload
